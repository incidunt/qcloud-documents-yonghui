1. 接口描述

本接口（DescribeDCDBRenewalPrice）用于在续费分布式数据库实例时，查询续费的价格。

2. 输入参数

以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表见公共请求参数。

参数名称 是否必选 类型 描述
Action 是 String 公共参数，本接口取值：DescribeDCDBRenewalPrice。
Version 是 String 公共参数，本接口取值：2018-04-11。
InstanceId 是 String 待续费的实例ID。形如：dcdbt-ow728lmc，可以通过 DescribeDCDBInstances 查询实例详情获得。
Period 否 Integer 续费时长，单位：月。不传则默认为1个月。

3. 输出参数

参数名称 类型 描述
OriginalPrice Integer 原价，单位：分。
Price Integer 实际价格，单位：分。受折扣等影响，可能和原价不同。
RequestId String 唯一请求ID，每次请求都会返回。定位问题时需要提供该次请求的RequestId。

4. 错误码

错误码 描述
InternalError.DbOperationFailed 查询数据库失败。
InternalError.QueryPriceFailed 查询价格失败。
InvalidParameterValue.SpecIdIllegal 找不到数据库实例对应的规格信息。

5. 示例

示例1 查询分布式数据库实例续费的价格

请求参数

https://dcdb.tencentcloudapi.com/?Action=DescribeDCDBRenewalPrice
&InstanceId=dcdvt-fdpjf5zh
&Period=1
&<公共请求参数>

返回参数

{
  "Response": {
    "OriginalPrice": 42240,
    "Price": 42240,
    "RequestId": "5b637383-8059-b0f8-3dcd-59fbdff262b6"
  }
}
