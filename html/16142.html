AddShardConfig

升级实例 -- 新增分片类型

名称 类型 是否必选 描述
ShardCount Integer 是 新增分片的数量
ShardMemory Integer 是 分片内存大小，单位 GB。
ShardStorage Integer 是 分片存储大小，单位 GB。

DCDBInstanceInfo

分布式数据库实例信息

名称 类型 是否必选 描述
InstanceId String 是 实例ID
InstanceName String 是 实例名称
AppId Integer 是 APPID
ProjectId Integer 是 项目ID
Region String 是 地域
Zone String 是 可用区
VpcId Integer 是 VPC数字ID
SubnetId Integer 是 Subnet数字ID
StatusDesc String 是 状态中文描述
Status Integer 是 状态
Vip String 是 内网IP
Vport Integer 是 内网端口
CreateTime Timestamp 是 创建时间
AutoRenewFlag Integer 是 自动续费标志
Memory Integer 是 内存大小，单位 GB。
Storage Integer 是 存储大小，单位 GB。
ShardCount Integer 是 分片个数
PeriodEndTime Timestamp 是 到期时间
IsolatedTimestamp Timestamp 是 隔离时间
Uin String 是 UIN
ShardDetail Array of ShardInfo 是 分片详情

Deal

订单信息

名称 类型 是否必选 描述
DealName String 是 订单号
OwnerUin String 是 所属账号
Count Integer 是 商品数量
FlowId Integer 是 关联的流程 Id，可用于查询流程执行状态。
InstanceIds Array of String 是 只有创建实例的订单会填充该字段，表示该订单创建的实例的 ID。

ExpandShardConfig

ShardInstanceIds Array of String 是 分片ID数组
ShardMemory Integer 是 分片内存大小，单位 GB。
ShardStorage Integer 是 分片存储大小，单位 GB。

RegionInfo

售卖可用区信息

名称 类型 是否必选 描述
Region String 是 地域英文ID
RegionId Integer 是 地域数字ID
RegionName String 是 地域中文名
ZoneList Array of ZonesInfo 是 可用区列表
AvailableChoice Array of ShardZoneChooseInfo 是 可选择的主可用区和从可用区

ShardInfo

分片信息

名称 类型 是否必选 描述
ShardInstanceId String 是 分片ID
ShardSerialId String 是 分片Set ID
Status Integer 是 状态
Createtime String 是 创建时间
Memory Integer 是 内存大小，单位 GB。
Storage Integer 是 存储大小，单位 GB。
ShardId Integer 是 分片数字ID

ShardZoneChooseInfo

分片节点可用区选择

名称 类型 是否必选 描述
MasterZone ZonesInfo 是 主可用区
SlaveZones Array of ZonesInfo 是 可选的从可用区

SpecConfig

按机型分类的规格配置

名称 类型 是否必选 描述
Machine String 是 规格机型
SpecConfigInfos Array of SpecConfigInfo 是 规格列表

SpecConfigInfo

实例可售卖规格详细信息，创建实例和扩容实例时 NodeCount、Memory 确定售卖规格，硬盘大小可用区间为[MinStorage,MaxStorage]。

名称 类型 是否必选 描述
NodeCount Integer 是 节点个数，2 表示一主一从，3 表示一主二从。
Memory Integer 是 内存大小，单位 GB。
MinStorage Integer 是 数据盘规格最小值，单位 GB。
MaxStorage Integer 是 数据盘规格最大值，单位 GB。
SuitInfo String 是 推荐的使用场景
Pid Integer 是 产品类型 Id
Qps Integer 是 最大 Qps 值

SplitRate Integer 是 数据切分比例
ShardMemory Integer 是 分片内存大小，单位 GB。
ShardStorage Integer 是 分片存储大小，单位 GB。

ZonesInfo

可用区信息
