1. 接口描述

本接口（DescribeDCDBPrice）用于在购买实例前，查询实例的价格。

2. 输入参数

以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表见公共请求参数。

参数名称 是否必选 类型 描述
Action 是 String 公共参数，本接口取值：DescribeDCDBPrice。
Version 是 String 公共参数，本接口取值：2018-04-11。
Zone 是 String 欲新购实例的可用区ID。
Count 是 Integer 欲购买实例的数量，目前只支持购买1个实例。
Period 是 Integer 欲购买的时长，单位：月。
ShardNodeCount 是 Integer 单个分片节点个数大小，可以通过 DescribeShardSpec
查询实例规格获得。
ShardMemory 是 Integer 分片内存大小，单位：GB，可以通过 DescribeShardSpec
查询实例规格获得。
ShardStorage 是 Integer 分片存储空间大小，单位：GB，可以通过 DescribeShardSpec
查询实例规格获得。
ShardCount 是 Integer 实例分片个数，可选范围2-8，可以通过升级实例进行新增分片到最多64个分片。

3. 输出参数

参数名称 类型 描述
OriginalPrice Integer 原价，单位：分。 原价，单位：分。
Price Integer 实际价格，单位：分。受折扣等影响，可能和原价不同。
RequestId String 唯一请求ID，每次请求都会返回。定位问题时需要提供该次请求的RequestId。

4. 错误码

错误码 描述
InternalError.QueryPriceFailed 查询价格失败。
InvalidParameterValue.SpecIdIllegal 找不到数据库实例对应的规格信息。

5. 示例

示例1 查询新购分布式数据库实例的价格

请求参数

https://dcdb.tencentcloudapi.com/?Action=CreateDCDBInstance
&ShardMemory=2
&ShardStorage=10
&ShardNodeCount=3
&ShardCount=2
&Zone=ap-guangzhou-2
&Count=1
&Period=1
&<公共请求参数>

返回参数

{
  "Response": {
    "OriginalPrice": 42240,
    "Price": 42240,
    "RequestId": "7e1000c2-190a-d0df-ff75-59fbdf5ff381"
  }
}
